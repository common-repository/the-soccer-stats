== Description ==

The Soccer Stats is an excellent WordPress plugin to be used with your football (soccer) team.

You may control players and matches season by season. The plugin automatically calculates stats for the players, creates match calendar list,  calculates opponent statistics and so on. Page templates are available for players, matches, seasons and opponents.

[Go to plugin web site for more info](http://thesoccerstats.wordpress.com)

== Installation ==

1. Download the plugin. Activate via Plugin-settings.
2. Ready to put some team data to the site!
